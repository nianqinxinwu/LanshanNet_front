## ⚠️ 注意事项

1. **API地址配置**
   - 确保 `$baseUrl` 配置正确
   - 生产环境和测试环境可能不同

2. **Token认证**
   - 所有API请求都需要在header中携带token
   - Token从 `uni.getStorageSync('token')` 获取

3. **文件上传**
   - 使用 `uni.uploadFile` 上传文件
   - 上传接口：`/api/xiluxc/Common/upload`
   - 返回格式：`{ code: 1, data: { url: "..." } }`

4. **金额格式化**
   - 显示时使用 `formatPrice()` 方法格式化为两位小数
   - 输入时使用 `type="digit"` 限制输入类型

5. **图片预览**
   - 图片使用 `uni.previewImage` 预览
   - PDF使用 `uni.downloadFile` + `uni.openDocument` 预览

## 📞 常见问题

### Q1: 页面跳转失败？
A: 检查 `pages.json` 是否正确添加了路由配置

### Q2: API请求401错误？
A: 检查token是否正确设置，是否已过期

### Q3: 文件上传失败？
A: 检查上传接口地址、文件大小限制、服务器权限

### Q4: 页面样式混乱？
A: 检查CSS类名是否与全局样式冲突，确保每个页面都有 `scoped` 属性
